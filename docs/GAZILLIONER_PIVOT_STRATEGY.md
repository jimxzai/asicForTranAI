### Step 1: Self-Host Inference (Week 1-2)

```
Current: Gazillioner → OpenAI API → Response
Future:  Gazillioner → Self-hosted 3.5-bit → Response (10x cheaper, 10x faster)
```

**Infrastructure**:
- 1x Nvidia A100 (or 2x RTX 4090)
- FastAPI wrapper around Fortran kernel
- Redis cache for common queries
- Cost: ~$2K/mo vs $20K/mo API costs
